Téléchargements

Raspberry Pi OS (anciennement Raspbian) est un système d’exploitation GNU/Linux spécialement conçu et optimisé pour la Raspberry Pi.
Sur cette page vous pouvez télécharger Raspberry Pi OS dans sa dernière version.

Raspberry Pi OS (anciennement Raspbian)

La dernière version de l’OS Raspberry Pi complet, plutôt adapté à la Raspberry Pi 3 et à la Raspberry Pi 4.
Identifiants : pi / raspberry
SHA-1 : d7a17375c7e84a89b2636d6152a204c6f1a16e03
Téléchargez Raspberry Pi OS

Raspberry Pi OS Lite (anciennement Raspbian Lite)

Il est possible de télécharger une version réduite de Raspberry Pi OS (notamment pour la Raspberry Pi A+ ou la Raspberry Pi Zéro).
Identifiants : pi / raspberry
SHA-1 : 10b2ce7323a65cb12b730f97ff79fd4c8150498f
Téléchargez Raspberry Pi OS Lite
